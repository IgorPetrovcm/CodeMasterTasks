# Ряд шагов для работы с Lab-8

+ Запустите **Docker Desktop**
+ В корневой директории лабораторной **Lab-8**, введите в консоль:
```
docker-compose build
```
Далее
```
docker-compose up
```
# Сводка
### Адрес
+ **Хост**: 127.0.0.1 или localhost
+ **Порт**: 5430
### Пользователь
+ **Username**: root
+ **Password**: root
